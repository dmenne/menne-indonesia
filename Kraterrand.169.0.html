Aufsatz 7. Schuljahr: Die Javanischen Vulkane - Sitze der Götter

Die Insel Java ist eine der vulkanreichsten Gegenden der Erde. In ganz Java gibt es 19 tätige Vulkane, in ganz Indonesien 55! Die Zahlen stimmen wohl nicht, aber damals gab es noch keine Links wie VolcanoWorld, wo man diese Information zusammen mit vielen Bildern finden kann. Besonders schöne Bilder vom Bromo gibt es auf GunungBagging . Davon entfällt ein einziger auf die Rieseninsel Borneo.
